The goal of this assignment is to create a style sheet that will make this page look different based on the size of the window. To do this we will use media queries. This page should display a 3 column paragraph with this text when the screen is more than 830 px, and the picture on the right side. When the browser is 480 to 830 pixels, the picture should be above this paragraph which should become a two column paragraph. Any size below 480 px should have the picture disappear and become single-column. Little hints

You can give the section a max-width so that it doesn't grow beyond a certain size. For media queries, you should use for example @media screen and ( min-width : 480px ) and ( max-width : 830px ). Columns for chrome are -webkit-column-count: 2. Rounded corners are done with border-radius
